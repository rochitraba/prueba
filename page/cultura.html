¿Qué buscamos?

Backend Node JS - Java Spring Developer
Es el responsable de implementar funcionalidades requeridas en Node JS y Java Spring y ejecutar diferentes tareas referidas a bases de datos relacionales.

Dev Back-End
Es el responsable de implementar funcionalidades requeridas en PHP o Java con Laravel o Spring respectivamente, ademas de ejecutar diferentes tipos de tareas referidas a bases de datos relacionales

Diseñador UI
Se encarga de definir la línea de diseño del producto y coordinar diseño de identidad de marca del cliente con el diseño del producto

Diseñador UX
Es la persona que entiende los User Stories Maps que le facilite el Product Manager y diseña el Modelo de Experiencias de usuario

Frontend React/Angular Dev
Es el especialista que se encarga de mostrar de manera amigable(experiencia agradable) para el usuario la información proporcionada por el Backend. y programar, es decir, traducir definición de estilo visual.

Mobile App Dev- React Native
Es la persona responsable de traducir las definiciones de diseño y estilo visual, realizados por el equipo de UX/UI, adaptándolos a diversos dispositivos y navegadores, ademas de mostrar de manera amigable(experiencia agradable) para el usuario la información proporcionada por el Backend.

Scrum Master
Es la persona que se encarga de liderar las siguientes reuniones: Daily, Review, Refinamiento, Planning y Retrospectiva y desbloquea impedimentos que surjan en los diferentes equipos
