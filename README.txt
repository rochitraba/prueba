Proyecto Final Folcode para el curso de Desarrollo Web de Coderhouse

Hi everybody! This is my first project of the Coderhouse Web Development course. 
The project is based on Folcode, a product factory located in San Juan - Argentina. 
The intention of creating the site was to remodel and improve both design and navigation of the website.

Hola a todos! Este es mi primer proyecto final del curso de Desarrollo Web de Coderhouse. 
El proyecto esta basado en la empresa sanjuanina Folcode, que se dedica al desarrollo de productos digitales. 
El propósito de crear el sitio fue armar un clon de la página original y mejorarla en diversos aspectos.
